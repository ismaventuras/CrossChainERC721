# CrossChainERC721

This project has been made for the Polkadot Global Series hackathon.

## How to deploy the contracts

There's a hardhat task called deployCrossChainERC721, to deploy the contract run it with the network argument to deploy it to your desired network

```bash
npx hardhat deployCrossChainERC721 --network moonbeam
```

Be sure to create your own .env.json file from the example .env file

## how to run the webapp

go to the client page, run npm install and then npm run dev

```bash
npm install
npm run dev
```
